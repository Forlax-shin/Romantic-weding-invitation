Eternity Wedding Theme

Thank you for purchasing Eternity, a minimalist seasonal wedding invitation template.

1. Folder Structure

  • index.html - Main template file
  • css/style.css - Custom styling and Seasonal variables
  • js/script.js - Countdown and Seasonal effects

2. How to Change Season

The theme supports 4 seasons (Spring, Summer, Autumn, Winter). To set a default season, open script.js and find the setSeason function call or use the Toggle button in the preview.

Note: The colors are controlled by the --hue variable in style.css.

3. Setting the Countdown Date

Open script.js and update the date in the targetDate variable:

const targetDate = new Date("December 31, 2025 23:59:59").getTime();
